写defaultProps有两种方法
1 在组件内部的使用static ,ES6写法(内部优先级大于外部优先级)
    static defaultProps = {
        name:　...
    };
2 在组件外部ES5,ES6都适用
    App.defaultProps =  {
        name: ...
    };



写state两种方法,都在组件内部
1 constructor(props){//第一种优先级大于第二种
    super(props);
    this.state={
        name: ...
    }
}
2 state = { name: ... }
